## What is xParse?

xParse turns PDFs, images, and Office documents into structured data. Get clean Markdown for LLMs, extract specific fields with JSON Schema, or retrieve document elements with coordinates.

Perfect for RAG pipelines, document automation, and data extraction workflows.

## What you can process

**Input formats:** PDF, images (PNG, JPG, WebP, BMP, TIFF), Word, Excel, PowerPoint, HTML, TXT, OFD, RTF

**Parse returns:**
- Markdown with preserved reading order
- Structured elements (headings, paragraphs, tables, images)
- Pixel coordinates for every element
- Tables with row/column structure
- Images, formulas, handwriting, form fields

**Extract returns:**
- JSON matching your schema
- Source citations with page numbers and bounding boxes
- Field-level confidence and coordinates

## Common use cases

- Build RAG systems with clean document input
- Extract invoice data for accounting automation
- Convert research papers to searchable Markdown
- Pull form fields from scanned documents
- Process receipts at scale
