Parse converts entire documents into clean, structured data—Markdown for LLMs and JSON elements with coordinates for visual grounding.

## What you'll get

<CardGroup cols={2}>
  <Card title="Markdown" icon="file-lines">
    Clean, AI-ready text with preserved structure—headings, paragraphs, tables, formulas, and reading order.
  </Card>
  
  <Card title="Structured Elements" icon="cube">
    JSON array of document components with types, page numbers, and bounding box coordinates.
  </Card>
</CardGroup>

**Use Parse for:**
- RAG pipelines and vector embeddings
- LLM input (GPT-4, Claude, etc.)
- Search indexing and knowledge bases
- Document Q&A with visual grounding

**When to use Parse vs Extract:**
- **Parse:** You need full document content with preserved structure
- **Extract:** You only need specific fields (invoice totals, form data, etc.)

## Before you start

1. Get API credentials from the [TextIn Console](https://www.textin.ai/console/dashboard/setting)
2. Prepare a document or use our [sample PDF](https://dllf.intsig.net/download/2025/Solution/textin/sample/pdf_to_markdown/sample_02.pdf)

<Info>
  **Supported formats:** PDF, images (PNG, JPG, WebP, BMP, TIFF), Word, Excel, PowerPoint, HTML, TXT, OFD, RTF  
  **File size limit:** 500MB  
  **Page limits:** Sync API (500 pages), Async API (2000+ pages)
</Info>

## Parse your first document

<Tabs>
  <Tab title="Python">
    ```bash
    # Install SDK
    pip install xparse-client
    ```
    
    ```python
    from xparse_client import XParseClient

    # Initialize client
    client = XParseClient(
        app_id="your-app-id",
        secret_code="your-secret-code"
    )

    # Parse document
    with open("document.pdf", "rb") as f:
        result = client.parse.run(file=f, filename="document.pdf")
    
    # Get Markdown
    print(result.markdown)
    
    # Access structured elements
    for element in result.elements:
        print(f"{element.type}: {element.text[:50]}")
    ```
  </Tab>

  <Tab title="Node.js">
    ```bash
    # Install SDK
    ```

    ```javascript
    import { XParseClient } from 'xparse-client';
    import fs from 'fs';

    // Initialize client
    const client = new XParseClient({
      appId: 'your-app-id',
      secretCode: 'your-secret-code'
    });

    // Parse document
    const result = await client.parse.run({
      file: fs.createReadStream('document.pdf'),
      filename: 'document.pdf'
    });
    
    // Get Markdown
    console.log(result.markdown);
    
    // Access structured elements
    result.elements.forEach(element => {
      console.log(`${element.type}: ${element.text.substring(0, 50)}`);
    });
    ```

## Understanding the output

### Markdown

Clean, structured content ready for LLMs:

```markdown
# Annual Report 2024

## Executive Summary

This report presents our key achievements...

### Financial Highlights

| Metric | 2024 | 2023 |
|--------|------|------|
| Revenue | $1.2M | $950K |
| Profit | $340K | $280K |
```

**Preserved elements:**
- Heading hierarchy (`#`, `##`, `###`)
- Paragraphs and lists
- Tables (HTML or Markdown format)
- Formulas in LaTeX
- Reading order

### Structured Elements

JSON array of document components:

```json
{
  "elements": [
    {
      "type": "Title",
      "text": "Annual Report 2024",
      "page_number": 1,
      "coordinates": [[0.1, 0.1], [0.9, 0.1], [0.9, 0.15], [0.1, 0.15]]
    },
    {
      "type": "Table",
      "text": "| Metric | 2024 | 2023 |...",
      "page_number": 2,
      "coordinates": [[0.2, 0.3], [0.8, 0.3], [0.8, 0.6], [0.2, 0.6]]
    }
  ]
}
```

**Element types:**
- `Title`, `NarrativeText`, `ListItem`
- `Table`, `TableCaption`
- `Image`, `ImageCaption`
- `Formula`, `PageHeader`, `PageFooter`

**Coordinates:**
- Four-point bounding boxes for every element
- Normalized to [0,1] range
- Convert to pixels with page dimensions

## Common use cases

### RAG pipelines

```python
# Parse document
result = client.parse.run(file=f, filename="knowledge_base.pdf")
markdown = result.markdown

# Send to LLM
response = openai.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": f"Summarize: {markdown}"}]
)
```

### Document Q&A with citations

```python
# Parse with coordinates
result = client.parse.run(file=f, filename="contract.pdf")

# Build citations
for element in result.elements:
    if "termination" in element.text.lower():
        print(f"Found on page {element.page_number}")
        print(f"Coordinates: {element.coordinates}")
```

### Knowledge base with table of contents

```python
from xparse_client import ParseConfig, Capabilities

# Get document structure
config = ParseConfig(capabilities=Capabilities(title_tree=True))
result = client.parse.run(file=f, config=config)

# Build table of contents
for heading in result.title_tree:
    print(f"{'  ' * heading.level}{heading.title} (p.{heading.page_number})")
```
