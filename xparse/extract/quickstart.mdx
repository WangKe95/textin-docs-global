## Create schema

Define the fields you want to extract using JSON Schema format:

```json
{
  "type": "object",
  "properties": {
    "invoice_number": {
      "type": "string",
      "description": "Invoice number"
    },
    "total_amount": {
      "type": "number",
      "description": "Total amount including tax"
    },
    "vendor_name": {
      "type": "string",
      "description": "Vendor or merchant name"
    }
  },
  "required": ["invoice_number", "total_amount", "vendor_name"]
}
```

**Schema structure:**
- `type`: Schema type (`"object"` at top level)
- `properties`: Collection of fields to extract
  - Each field has:
    - `type`: Field data type (see types below)
    - `description`: What to extract (helps AI understand)
    - `enum`: List of allowed values (for enum type)
    - `items`: Structure for array elements (for array type)
- `required`: List of required fields (also defines field order)

**Supported field types:**
- `string` - Text
- `number` - Numeric (int or float)
- `integer` - Integer only
- `enum` - Predefined choices
- `object` - Nested fields
- `array` - List of items

**Type can include null:**
```json
"type": ["string", "null"]  // Field can be string or null
```
Even without null, the API returns null for missing fields.

<Tip>
  **Use the visual builder:** Create schemas easily with our [web interface](https://www.textin.ai/console/recognition/robot_markdown_beta?service=pdf_to_markdown&tab=llmExtract). See [Schema Guide](/xparse/extract/schema-guide) for details.
</Tip>

<Warning>
  **Schema limits:**
  - Maximum **3 levels** of nesting (object/array depth)
  - Maximum **100 leaf fields** (lowest-level fields that hold actual values)
</Warning>

## Call API

<Tabs>
  <Tab title="Python">
    ```bash
    # Install SDK
    pip install xparse-client
    ```
    
    ```python
    from xparse_client import XParseClient
    import json

    # Initialize client
    client = XParseClient(
        app_id="your-app-id",
        secret_code="your-secret-code"
    )

    # Load schema
    schema = json.loads(open("invoice_schema.json").read())

    # Extract data
    with open("invoice.pdf", "rb") as f:
        result = client.extract.run(
            file=f,
            schema=schema
        )
    
    # Get extracted data
    print(result.extracted_data)
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    curl -X POST "https://api.textin.com/ai/service/v3/entity_extraction" \
      -H "x-ti-app-id: YOUR_APP_ID" \
      -H "x-ti-secret-code: YOUR_SECRET_CODE" \
      -H "Content-Type: application/json" \
      -d '{
        "file": {
          "file_url": "https://example.com/invoice.pdf"
        },
        "schema": {
          "type": "object",
          "properties": {
            "invoice_number": {
              "type": "string",
              "description": "Invoice number"
            },
            "total_amount": {
              "type": "number",
              "description": "Total amount including tax"
            }
          }
        }
      }'
    ```
  </Tab>

  <Tab title="Node.js">
    ```bash
    # Install SDK
    npm install xparse-client
    ```
    
    ```javascript
    import { XParseClient } from 'xparse-client';
    import fs from 'fs';

    // Initialize client
    const client = new XParseClient({
      appId: 'your-app-id',
      secretCode: 'your-secret-code'
    });

    // Load schema
    const schema = JSON.parse(fs.readFileSync('invoice_schema.json'));

    // Extract data
    const result = await client.extract.run({
      file: fs.createReadStream('invoice.pdf'),
      schema: schema
    });
    
    // Get extracted data
    console.log(result.extractedData);
    ```

## Response

The API returns structured JSON matching your schema:

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "extracted_data": {
      "invoice_number": "INV-2024-001",
      "total_amount": 1250.00,
      "vendor_name": "Acme Corp"
    },
    "citations": {
      "invoice_number": {
        "value": "INV-2024-001",
        "bounding_regions": [
          {
            "page_number": 1,
            "position": [137, 599, 1129, 599, 1129, 625, 182, 625],
            "text": "INV-2024-001"
          }
        ]
      }
    }
  }
}
```

**What you get:**
- `extracted_data`: JSON matching your schema
- `citations`: Source locations with page numbers and pixel coordinates

## Additional options

You can customize extraction with additional parameters:

**parse_options** - Document preprocessing:
```json
{
  "parse_options": {
    "crop_dewarp": 1,        // Auto-crop and straighten (0 or 1)
    "get_image": "both"      // Image extraction: "none", "page", "objects", "both"
  }
}
```

**extract_options** - Extraction behavior:
```json
{
  "extract_options": {
    "generate_citations": true,  // Include source locations
    "stamp": true                // Extract stamp/seal information
  }
}
```

**Full example with options:**
```python
result = client.extract.run(
    file=f,
    schema=schema,
    parse_options={
        "crop_dewarp": 1,
        "get_image": "both"
    },
    extract_options={
        "generate_citations": True,
        "stamp": True
    }
)
```
