Parse a document and get AI-ready Markdown in minutes.

## Before you start

<Steps>
  <Step title="Get API credentials">
    Sign up and get your `x-ti-app-id` and `x-ti-secret-code` from the [TextIn Console](https://www.textin.ai/console/dashboard/setting).
    
    See [Authentication](/xparse/authentication) for detailed instructions.
  </Step>

  <Step title="Prepare a sample file">
    Use your own document or download our sample: [Sample Document.pdf](https://dllf.intsig.net/download/2025/Solution/textin/sample/pdf_to_markdown/sample_02.pdf)
    
    <Info>
      **Supported formats**: PDF, images (PNG, JPG, WebP, BMP, TIFF), Word, Excel, PowerPoint, HTML, TXT, OFD, RTF
      
      **File size limit**: 500MB
    </Info>
  </Step>
</Steps>

## Step 1: Install SDK

<Tabs>
  <Tab title="Python">
    <CodeGroup>
    ```bash pip
    pip install xparse-client
    ```

    ```bash uv
    uv add xparse-client
    ```
    </CodeGroup>

    Set your credentials as environment variables:

    ```bash
    export TEXTIN_APP_ID="your-app-id"
    export TEXTIN_SECRET_CODE="your-secret-code"
    ```
  </Tab>

  <Tab title="cURL">
    No installation required. Use cURL directly with your credentials.
  </Tab>

  <Tab title="Node.js">
    ```bash
    npm install xparse-client
    ```

    Set your credentials as environment variables:

    ```bash
    export TEXTIN_APP_ID="your-app-id"
    export TEXTIN_SECRET_CODE="your-secret-code"
    ```
  </Tab>
</Tabs>

## Step 2: Parse a document

<Tabs>
  <Tab title="Python">
    ```python
    from xparse_client import XParseClient

    # Initialize client (reads from environment variables)
    client = XParseClient()

    # Parse document
    with open("document.pdf", "rb") as f:
        result = client.parse.run(
            file=f,
            filename="document.pdf"
        )
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    curl -X POST "https://api.textin.ai/api/v1/xparse/parse/sync" \
      -H "x-ti-app-id: YOUR_APP_ID" \
      -H "x-ti-secret-code: YOUR_SECRET_CODE" \
      -F "file=@document.pdf"
    ```
  </Tab>

  <Tab title="Node.js">
    ```javascript
    import { XParseClient } from 'xparse-client';
    import fs from 'fs';

    // Initialize client (reads from environment variables)
    const client = new XParseClient();

    // Parse document
    const result = await client.parse.run({
      file: fs.createReadStream('document.pdf'),
      filename: 'document.pdf'
    });
    ```
  </Tab>
</Tabs>

### Example response

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "file_id": "doc_7f3a2b",
    "success_count": 5,

    "elements": [
      {
        "type": "Title",

}
```

## Step 3: Use the Markdown

<Tabs>
  <Tab title="Python">
    ```python
    # Get the Markdown content
    markdown = result.markdown

    # Use it for RAG, embeddings, or search
    print(markdown)

    # Or save it to a file
    with open("output.md", "w", encoding="utf-8") as f:
        f.write(markdown)
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    # Extract markdown from the response using jq
    curl -X POST "https://api.textin.ai/api/v1/xparse/parse/sync" \
      -H "x-ti-app-id: YOUR_APP_ID" \
      -H "x-ti-secret-code: YOUR_SECRET_CODE" \
      -F "file=@document.pdf" \
      | jq -r '.data.markdown' > output.md
    ```
  </Tab>

  <Tab title="Node.js">
    ```javascript
    // Get the Markdown content
    const markdown = result.markdown;

    // Use it for RAG, embeddings, or search
    console.log(markdown);

    // Or save it to a file
    fs.writeFileSync('output.md', markdown, 'utf-8');
    ```
  </Tab>
</Tabs>

**What you can do with the Markdown:**
- Send it to LLMs for summarization or Q&A
- Generate embeddings for vector search
- Index it in your search engine
- Build a RAG pipeline

## What's included

Every parsing response includes:

<CardGroup cols={2}>
  <Card title="Markdown" icon="file-lines">
    Clean, structured content ready for LLMs. Preserves headings, tables, lists, and reading order.
  </Card>
  
  <Card title="Structured Elements" icon="cube">
    JSON with typed elements (headings, paragraphs, tables, images) and their coordinates.
  </Card>
  
  <Card title="Tables" icon="table">
    Available in HTML, Markdown, or structured JSON format with row/column information.
  </Card>
  
  <Card title="Coordinates" icon="location-dot">
    Bounding boxes for every element, useful for visual grounding and citations.
  </Card>
</CardGroup>
